    Halaman ini dibuat untuk memberikan informasi tentang siapa yang membuat situs web ini dan untuk apa situs web ini digunakan. Situs web ini dibuat oleh Bima Sanjaya, lahir pada tanggal 7 April 2002 di Kabupaten Semarang yang terkenal dengan keindahan alamnya. Bima Sanjaya adalah seorang mahasiswa yang sedang menempuh studinya di Pendidikan Teknik Informatika dan Komputer, jurusan Teknik Elektro, Fakultas Teknik, Universitas Negeri Semarang.

    Saya membuat situs web ini sebagai salah satu syarat kelulusan dalam menyelesaikan skripsi yang berjudul "Teks Summarization". Situs web ini merupakan hasil dari pemikiran dan kerja keras Saya dalam menyelesaikan salah satu syarat wajib untuk kelulusan. Saya berharap bahwa situs web ini dapat memberikan manfaat bagi para pengguna yang membutuhkan fitur teks summarization.

    Petunjuk penggunaan situs web ini juga tersedia pada halaman ini. Saya berharap bahwa para pengguna dapat menggunakan situs web ini dengan mudah dan nyaman. Jika ada masalah atau kendala dalam menggunakan situs web ini, pengguna dapat menghubungi Saya melalui kontak yang tersedia pada halaman ini.

    Saya berharap bahwa situs web ini dapat memberikan solusi bagi para pengguna yang membutuhkan fitur teks summarization. Saya juga berharap bahwa situs web ini dapat menjadi salah satu sumber informasi yang berguna bagi para pengguna.
